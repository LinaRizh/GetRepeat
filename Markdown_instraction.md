## Списки

Для добавления ненумерованного списка необходимо пункты выделить (*).
Например, вот  так:
* Элемент 1
* Элемент 2
* Элемент 3

Для добавления нумерованного списка необходимо пункты пронумеровать.
Например, вот  так:
1. Элемент 1
2. Элемент 2
